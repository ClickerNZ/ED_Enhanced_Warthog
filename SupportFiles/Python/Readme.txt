The included python scripts require python for windows and watchdog to be installed.

Python: https://www.python.org/downloads/

Watchdog: from the command line enter: pip install watchdog

To run the script, either;

1.
a. Open a command prompt where you copied the package
b. enter: cd SupportFiles\Powershell
c. enter: python WatchJournal.python

or 

2. 
a. using windows exploerer, navigate to where you copied the script package
b. navigate to SupportFiles, then Python
c. Create a shortcut on your desktop of the WatchJournal.cmd batch file
d. Right-click the new shortcut and choose properties
e. Click Advanced and click the Run as Administrator checkbox.
f. Click Ok and Ok again.
